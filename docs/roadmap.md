# Roadmap

## Near-term
- Add OpenAPI spec generation and publish Swagger UI.
- Add API token scopes and optional MFA-ready auth extension points.
- Add bulk document metadata import endpoint.

## Mid-term
- Add background jobs for heavier audit/report exports.
- Add optional object storage integration (S3-compatible) for binary file streams.
- Add webhook delivery for selected audit events.

## Long-term
- Add cross-organization reporting views for multi-tenant operators.
- Add immutable export bundles for compliance review workflows.
- Add blue/green deployment examples with zero-downtime migrations.
